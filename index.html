Icarus Display Unit


The current version of my software product which is called Icarus is being developed now. This software has a modern frontend framework which works on the web. Attractive to look at using twitter bootstrap. It uses the latest JavaScript technologies AngularJS for interactivity. The service offered is one for analysing large data sets. If you have big data we offer an attractive user interface for searching, tabulating and storing large datasets. The test case is on genomics data, the works of William Shakespeare and astrophysics data.

The software runs on a raspberry pi so it is cheap, much cheaper than buyer a new pc. This means that multiple individual workstations can be purchased. We use cutting eadge natural text processing techniques running on the linux kernal on the raspberry pi to deliver low cost, high performance computing tailor made to scientific requirements.

Metamaterials were described as one of the top 10 scientific breakthroughs of the last decade by Science magazine. Metamaterials are a new paradigm in science. A modern day scientific revolution where a slight change in perception, as Einsteins slight reformulation of the laws of relativity has led to a whole new field in Physics.

The technology has immediate applications to optics and antenna technology. Kymeta use metamaterials for flat space communication dishes. Kymeta has so far received 2 rounds of £50m funding from Bill Gates and was for two years running voted in the top 50 disruptive technologies by techcrunch disrupt.
